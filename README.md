# 🌸 WindowTotalCounter

SillyTavern 插件：統計聊天字數
A useful plugin for chat word count within SillyTavern

## ✨ 功能特色

- 🌸 **櫻花按鈕** — 精緻的粉色五瓣櫻花 SVG 圖示，取代傳統文字按鈕
- 🔄 **持續旋轉動畫** — 櫻花緩慢旋轉，hover 時加速，拖動中更快
- 💫 **呼吸光暈** — 粉色脈動光暈效果，讓按鈕更加醒目
- 🖱️ **可自由拖動** — 按鈕可拖到畫面任意位置，不再擋住其他按鍵
- 💾 **位置記憶** — 拖動後自動記住位置，下次載入自動恢復
- 🌸 **花瓣飄落特效** — 點擊開啟面板時，會從按鈕噴出飄落的櫻花瓣
- 📊 **即時統計** — 顯示當前聊天的訊息數量與總字符數
- 📱 **觸控支援** — 支援手機/平板觸控拖動
